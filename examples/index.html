Bootstrap-session-timeout

Basic Usage

Shows the warning dialog after click the demo button below. If user takes no action, browser is redirected to logout_redirect_url.

            $(document).ready(function () { 
                $.timeoutDialog({
                    timeout: 120 // seconds
                    ,countdown: 120 // seconds
                    ,logout_redirect_url: 'https://github.com/stephenwomack/timeout-dialog-bootstrap.js'
                    ,restart_on_yes: false
                });
            });
