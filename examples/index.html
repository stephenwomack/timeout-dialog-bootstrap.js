Bootstrap-session-timeout

Basic Usage

Shows the warning dialog after 5 seconds. If user takes no action, browser is redirected to redirUrl.

            $(document).ready(function () { 
                $.timeoutDialog({
                    timeout: 125 // seconds
                    ,countdown: 120 // seconds
                    ,keep_alive_url: ''
                    ,logout_redirect_url: 'https://github.com/stephenwomack/timeout-dialog-bootstrap.js'
                    ,restart_on_yes: false
                });
            });
